Artificial Intelligence simply meaning is making its own decision without human interaction. Machine Learning and Deep Learning are a subset of Artificial Intelligence and the Final Goal of these are Making AI Application.

Machine Learning

Machine Learning can be divide into 3 parts. Supervised ML, Unsupervised ML, Reinforcement ML are 3 types of ML.

  1. Supervised ML
    • Supervised Machine Learning can divide into two parts as Classification and Regression. In Supervised Learning, we have Labeled data ( as Input Data and Output Data ) if Output data has a continuous value this will belongs to the Regression case and otherwise, it belongs to Classification.
  2. Unsupervised ML
    • Unsupervised Learning not having labeled data clustering is a use case of unsupervised learning.
  3. Reinforcement ML
    • We can call Reinforcement Learning as semi-supervised Learning. in childhood parents advise their children do this do not do this if you do this will harm your life but after childhood, they do not need advice they can understand by themself Reinforcement learning behave like this and that's why Reinfrorcement learning called as Semi-Supervised Learning
