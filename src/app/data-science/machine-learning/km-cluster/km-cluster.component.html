K Means Clustering

Maths Behind K-Means Clustering



First, this algorithm tries to start with K Centroids by putting them at a random place,

  • Start with K centroids by putting them at random place
  • Compute distance of every point from centroid and cluster them accordingly
  • Adjust centroids so that they become center of gravity for given cluster
  • Again re-cluster every poits based on their distance with centroid
  • Again adjust centroids
  • Recompute clusters and repeat this till data points stop chhanging clusters
How to determine correct number of clusters (k)

We can use SSE for each cluster and using L bow method we can choose suitable k Value
