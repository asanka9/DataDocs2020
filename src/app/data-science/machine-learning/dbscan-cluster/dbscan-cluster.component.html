DBSCAN Algorithom


Maths Behiand DBSCAN Algorithom

Here we have 4 different main components

  • Epsilon
  • Minimum Points
  • Core Points
  • Border Points
  • Noise Points

Here we consider p point and draw a circular boundary around P according to radius value. ere Radius value is Epsilon Value. if This circular boundary contains at least Minimum points this p consider as a core point otherwise if the circular boundary contain at least one point this P point consider as a Border point and if the circular boundary does not contain any point P consider as Noise Point

These Noise Points are Like Outliers and Noise points not assigned to a cluster because this algorithm works very well with Outliers

Advantage of DBSCAN
  • Is greater at separating cluster of high-density Vs Law density within a given data set.
  • Is great with handling outliers within the dataset
Disadvantages of DBSCAN
  • Does not work very well with dealing with clusters which has a similar density
  • Struggles with high dimensionality data
