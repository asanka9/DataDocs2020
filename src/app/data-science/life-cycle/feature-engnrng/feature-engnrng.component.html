What is Feature Engineering

One Hot Encoding

how we can perform One hot Encoding for Multi Categorical Features

If we have a low number of categorical features it is not a problem for applying one-hot encoding but the number of categorical features increases and after applying one-hot encoding, it is a reason To increase the number of variables, to prevent this we can get the most frequent categorical features for applying one-hot encoding

Advantages:
  • Straightforward to implement
  • Does not expand massively the number of columns in the data set
Disadvantages:
  • Does not keep any information about the ignored label
