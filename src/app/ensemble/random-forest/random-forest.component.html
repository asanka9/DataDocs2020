Random forest algorithom

we can use a random forest algorithm for regression and classification of both cases. random forest is a bagging technique, in the random forest we use multiple decision trees. in random forest we pickup some sample of raws and some sample of columns in our data set randomly, this is also called as raw sampling and feature sampling. this raw sampling and feature sampling happen with replacemenet


In classification cases use mode of outputs, but in regression cases get mean of all ouputs

How random forest works with overfitting

In random forest use multiple decision trees. considering a single decision tree it tends to train data set very well therefore overfitting problems may happen using a single decision tree. but in a random forest using multiple decision trees with raw and feature sampling. therefore random forest prevent the overfitting problem

Advantages of random forest
  • Doesn't overfit
  • Less parameter tuning required
  • Decision tree handle both continous and categorical variables
  • No feature scaling required
  • Suitable for any ML case
  • Robust to outliers
Performance metrics
  • Regression
    • R2,Adjusted R2
    • MSE,RMSE,MAE
  • Classification
    • Confusion Matrix
    • Precision,Recall, F1 score
Disadvantages of Random Forest

Biased With features having many categories (Biased in multiclass classification problems towards more frequent classes.)
